# Riddle API --- ### GET /api/v1/config/{type} **Config Types:** {filter|user|correlator} - **filter:** Returns an JSON array of filter configs - **user:** Returns the UI config - **correlator:** Return the correlator config JSON **JSON structure: user** ``` { "chain":[], "toolbar":[ {"name":"filter","order":1}, {"name":"dedub","order":2}, {"name":"correlator","order":3}, {"name":"unique filter","order":4} ] } ``` --- ### POST /api/v1/config/{type} **Config Types:** {filter|user} - **filter:** Overwrite Filter Config - **user:** Overwrite UI Config - **correlator:** Overwrite Correlator Config **JSON structure: filter** ``` [ { "name":"xyz", "threshold":10, "treshold-top":55, "unit":"MosDev", "active":false }, { "name":"no2", "threshold":20, "threshold-top":500, "unit":"love babis", "active":false } ] ``` ``` curl -H "Content-Type: application/json" -X POST -d '{"chain":[{"name":"filter","order":1}], "toolbar":[{"name":"dedub","order":2},{"name":"correlator","order":3},{"name":"unique filter","order":4}]}' http://riddle-api.mybluemix.net/api/v1/config/filter ``` **JSON structure: correlator** ``` { "matcher_id":"id", "matcher_value":"15", "name":"test", "pitch":1, "time_treshold":5, "timestamp":"identifier", "timestamp_format":"234Tdsfas" } ``` ``` curl -H "Content-Type: application/json" -X POST -d '{"matcher_id":"id","matcher_value":"15","name":"test","pitch":1,"time_treshold":5,"timestamp":"identifier","timestamp_format":"234Tdsfas"}` http://riddle-api.mybluemix.net/api/v1/config/correlator ```
